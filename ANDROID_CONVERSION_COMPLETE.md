## How to Build APK

### Prerequisites
1. Install [Android Studio](https://developer.android.com/studio)
2. Install JDK 17 or higher
3. Ensure Node.js is installed

### Quick Build (3 commands)
```bash
npm install                # Install dependencies
npm run android:init       # Create Android project (first time only)
npm run android:run        # Build, sync, and open in Android Studio
```

### Build APK in Android Studio
1. Wait for Gradle sync to complete
2. Click **Build > Build Bundle(s) / APK(s) > Build APK(s)**
3. APK location: `android/app/build/outputs/apk/debug/app-debug.apk`

### Install on Device
```bash
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

## Google Play Store Submission

### 1. Create Release Keystore
```bash
keytool -genkey -v -keystore clipboard-max.keystore \
  -alias clipboard-max -keyalg RSA -keysize 2048 -validity 10000
```

### 2. Update Configuration
Edit `capacitor.config.ts`:
```typescript
android: {
  buildOptions: {
    keystorePath: './clipboard-max.keystore',
    keystoreAlias: 'clipboard-max'
  }
}
```

### 3. Build Signed Bundle
In Android Studio:
1. **Build > Generate Signed Bundle / APK**
2. Select **Android App Bundle** (AAB)
3. Choose your keystore
4. Build release

### 4. Submit to Play Store
1. Create Google Play Developer account ($25 one-time)
2. Create new app in Play Console
3. Upload AAB file
4. Add screenshots, description, icon
5. Submit for review

## Technical Details

### Clipboard API Migration
**Before (Web):**
```typescript
await navigator.clipboard.writeText(text);
const text = await navigator.clipboard.readText();
```

**After (Capacitor):**
```typescript
import { Clipboard } from '@capacitor/clipboard';

await Clipboard.write({ string: text });
const { value } = await Clipboard.read();
```

### Back Button Handler
```typescript
import { App as CapApp } from '@capacitor/app';

useEffect(() => {
  const handler = CapApp.addListener('backButton', () => {
    // Handle navigation or exit app
  });
  return () => handler.then(l => l.remove());
```

### Common Issues

**"Gradle sync failed"**
- Ensure JDK 17+ is installed and selected in Android Studio
- Check internet connection (Gradle downloads dependencies)
- Update Android Studio to latest version

**"APK won't install"**
- Enable "Install from Unknown Sources" on device
- Use `adb install -r` flag to reinstall

**"Capacitor not found"**
- Run `npm install` to get all dependencies
- Check that `node_modules/@capacitor` exists
